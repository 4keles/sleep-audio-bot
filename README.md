# sleep-audio-botü


Application Name: Sleep Audio Factory
Description:
A personal automation tool that programmatically searches and downloads CC0-licensed nature sounds and ambient audio from Freesound to build sleep and relaxation video content. The tool queries specific sound categories (rain, fire, ocean waves, wind, forest) using the text search API, filters strictly for CC0 license, and downloads preview MP3s for local archival. No sounds are redistributed or re-uploaded. All content is used in original video productions with full attribution logged. This is a self-hosted, non-commercial personal project running on a private Linux server.
